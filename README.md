Это бот-переводчик в вк, который обладает следующими функциями:
- Быстрый перевод с любого языка на русский в одно сообщение
- Перевод в два сообщения с одного языка на любой другой
- Определитель языка по тексту на этом языке
Бот использует API яндекс переводчика для функций перевода и определения языка, а также vk api чтобы функционировать в соц.сети ВКонтакте
Также бот распознает приветствие на любом языке и благодарность, чтобы сохранять контекст диалог.
Написать боту можно здесь https://vk.com/club172186151. Надо перейти в сообщения группы и нажать начать/написать что-угодною
